== Description ==
一款多功能插件,集成了微信工众号,服务号等功能, 支持微信支付,微薄登录,QQ和短信登录功能. 支付宝支付, 微信支付功能.只要在后台设置 appid 即可立即使用这些社交功能.
= Features = 
* 支持微信,QQ,微博登录中国最流行的社交软件登录
* 支持手机+验证码快速登录（需要自行购买SP短信服务）
* 支持与WordPress账号绑定和解绑，可以同时绑定多个社交账号
= GitHub: =

* Project url in GitHub: https://github.com/kuibobo/GamPress/

== Installation ==

1. 插件上传到plugin目录,并到WordPress后台激活
2. 在GamPress设置模块,设置相关appid和secret
